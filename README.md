# Custom Tetris
Tetris game with editable pieces

## Instructions
Install the pygame library by typing this into the terminal:
```
pip3 install pygame
```
Once installed, navigate to the folder with the file inside and type:
```
python3 blockgame.py
```
Have fun playing!
